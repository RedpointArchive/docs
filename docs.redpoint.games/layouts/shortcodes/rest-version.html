Endpoints

All HiveMP services offer a production endpoint and a development endpoint. The production endpoint has an uptime SLA as outlined in your plan, while no uptime is guaranteed for the development endpoint.

The development endpoint may receive updates and new features earlier than the production endpoint. You can use the development endpoint during your own testing to keep data separate to production.
